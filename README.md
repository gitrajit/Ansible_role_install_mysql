# Ansible_role_install_mysql

This is an example to show how to install Mysql server using ansible roles.


```
.
├── ansible.cfg
├── dev_env
├── roles
│   └── mysql
│       ├── defaults
│       │   └── main.yml
│       ├── handlers
│       │   └── main.yml
│       ├── meta
│       │   └── main.yml
│       ├── README.md
│       ├── tasks
│       │   └── main.yml
│       ├── tests
│       │   ├── inventory
│       │   └── test.yml
│       └── vars
│           └── main.yml
└── site.yml
```
